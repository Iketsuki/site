條件判斷：做決定的能力

條件判斷就像是我們日常生活中做決定的過程。例如：如果下雨了，我就帶傘；如果沒下雨，我就不帶傘。

生活中的條件判斷：

  • 如果肚子餓了，就去吃飯
  • 如果天氣熱，就開冷氣
  • 如果考試及格，就獲得獎勵
# Python 中的條件判斷
分數 = 85

if 分數 >= 90:
    print("優秀！獲得 A 等級")
elif 分數 >= 80:
    print("良好！獲得 B 等級")
elif 分數 >= 70:
    print("尚可！獲得 C 等級")
elif 分數 >= 60:
    print("及格！獲得 D 等級")
else:
    print("需要加油！獲得 F 等級")

比較運算符：做比較

等於 (==)

檢查兩個值是否相等

5 == 5  # True

不等於 (!=)

檢查兩個值是否不相等

5 != 3  # True

大於 (>)

檢查左邊的值是否大於右邊

7 > 3  # True

小於 (<)

檢查左邊的值是否小於右邊

2 < 8  # True

大於等於 (>=)

檢查左邊的值是否大於或等於右邊

5 >= 5  # True

小於等於 (<=)

檢查左邊的值是否小於或等於右邊

4 <= 4  # True

迴圈：重複的力量

迴圈就像是重複做同一件事情，直到達成目標為止。就像是你在練習寫字時，會重複寫同一個字多次，直到寫得漂亮為止。

生活中的迴圈：

  • 每天刷牙
  • 每週上學五天
  • 重複練習鋼琴直到熟練

for 迴圈：已知次數的重複

當你知道要重複幾次時，使用 for 迴圈。就像是你要練習寫一個字 10 次。

# 使用 for 迴圈印出 1 到 5
for 數字 in range(1, 6):
    print("這是第", 數字, "次重複")

# 輸出:
# 這是第 1 次重複
# 這是第 2 次重複
# 這是第 3 次重複
# 這是第 4 次重複
# 這是第 5 次重複

while 迴圈：條件式的重複

當你不確定要重複幾次，但知道什麼條件下要停止時，使用 while 迴圈。就像是你要練習鋼琴直到彈得完美為止。

# 使用 while 迴圈計算 1+2+3+...+10 的總和
總和 = 0
數字 = 1

while 數字 <= 10:
    總和 = 總和 + 數字
    數字 = 數字 + 1

print("1 到 10 的總和是:", 總和)  # 輸出: 1 到 10 的總和是: 55

函數：可重複使用的工具

函數就像是一個特殊的工具，你可以隨時拿出來使用。就像是廚房中的食物調理機，你可以隨時使用它來處理食材。

生活中的函數：

  • 食譜：輸入食材，輸出美食
  • 計算機：輸入數字，輸出計算結果
  • 洗衣機：輸入髒衣服，輸出乾淨衣服
# 定義一個計算兩數之和的函數
def 計算總和(數字1, 數字2):
    結果 = 數字1 + 數字2
    return 結果

# 呼叫函數
a = 5
b = 3
總和 = 計算總和(a, b)
print(a, "+", b, "=", 總和)  # 輸出: 5 + 3 = 8

函數名稱

參數

就像是工具需要的原料或設定

函數內容

就像是工具的運作方式

返回值

就像是工具產生的結果
