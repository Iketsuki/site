Reverse Search & Slicing

Finding the last space by scanning backwards and using s[0 : i] to capture the prefix.

Step 0: Setup

full_name = "Tai Man Chan"
n = (full_name)
space_idx = 0

Step 1: Scan from the Back

The first space we find from the right is the last space of the string.

for i in range(, , ):
if full_name[i] == " ":
space_idx = i
break # Stop at the first space we find

Step 2: Capture "Tai Man"

Use the slicing pattern [start : stop] to get everything before the space.

first_middle = full_name[ : ]
Expected first_middle
"Tai Man"

Step 3: Capture "Chan"

Capture from after the space to the end of the string.

last_name = full_name[space_idx + 1 : ]
Expected last_name
"Chan"

Step 4: The Result

swapped = last_name + " " + first_middle
print(swapped)
Final Output
"Chan Tai Man"

Algorithm Summary

Concept Python Syntax Why it matters?
Reverse Loop range(n-1, -1, -1) Finds the right-most match first.
Prefix Slice string[0 : i] Captures all chars before index i.
Suffix Slice string[i+1 : n] Captures all chars after index i.
