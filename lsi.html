Finding the Index

Given the following Python list:

fruits = ["apple", "banana", "cherry", "date"]

Task: Find the index of "cherry" and print its position.

Step 0: Variable Definition

We will track the position (index) found:

Item to find: target = "cherry"
Found status: found = False
Storage for index: pos = -1

Step 1: Computer Index vs Human Count

"cherry" is the 3rd item in the list. What are its Python index and its human position?

Python Index: (Starts at 0)
Human Position: (Starts at 1)
