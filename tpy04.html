Lesson 1: The Robot Arms (Arithmetic Operators)

ü™ù Section 1: Specialized Robot Arms

Arithmetic Operators are special symbols that command the computer to perform calculations. Think of them as **specialized Robot Arms** on a construction site, each designed for a specific task:

  • **: **Stacking Crane** (Exponentiation)
  • +: **Joining Clamp** (Addition)
  • -: **Chisel** (Subtraction)
  • *: **Cloning Machine** (Multiplication)
  • /: **Laser Cutter** (Division, includes decimals)
  • //: **Slicer** (Floor Division, discards decimals)
  • %: **Scrap Collector** (Modulo, saves only the remainder)
