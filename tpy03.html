1. üé£ The Hook: How to Speak

The print() function takes the data inside its parentheses and announces it to the user on the screen. It is like a Megaphone.

Key Rules:

  • Text (String): Must be inside quotes. The megaphone speaks the exact message.
  • Variables: The megaphone takes the value from the box and speaks that value.
  • Separators (Commas): If you separate items with a comma (e.g., `print(A, B)`), the megaphone automatically adds a space between them.
  • Newline: After every full `print()` command, the screen cursor drops down one line.
