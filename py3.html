迴圈與陣列遍歷

在 Python 程式設計中，迴圈是一種重要的控制結構，用於重複執行一段程式碼。當我們需要處理陣列（列表）中的元素時，迴圈就顯得尤為重要。

本教學將介紹兩種主要的迴圈類型：while 迴圈和for 迴圈，並說明如何使用它們來遍歷陣列，查找或修改其中的值。

While 迴圈

While 迴圈會在條件為真時重複執行程式碼區塊。它適合用於不確定需要重複多少次的情況。

基本語法

while 條件: # 當條件為真時執行的程式碼 程式碼區塊

使用 while 迴圈遍歷陣列

numbers = [1, 2, 3, 4, 5] i = 0 while i < len(numbers): print(numbers[i]) i += 1
注意：Python 使用縮排（通常是 4 個空格）來表示程式碼區塊。 在 while 迴圈中，所有縮排的程式碼都會在條件為真時重複執行。

視覺化演示

numbers = [1, 2, 3, 4, 5] i = 0 while i < len(numbers): print(numbers[i]) i += 1

While 迴圈的特點

  • 適合用於不確定迭代次數的情況
  • 必須手動更新計數器或條件，否則可能導致無限迴圈
  • 可以使用 break 提前退出迴圈
  • 可以使用 continue 跳過當前迭代

實際應用：在陣列中查找特定值

numbers = [12, 45, 7, 23, 56] target = 23 i = 0 found = False while i < len(numbers): if numbers[i] == target: print(f"找到目標值 {target} 在索引 {i}") found = True break i += 1 if not found: print(f"未找到目標值 {target}")

For 迴圈

For 迴圈用於遍歷可迭代對象（如列表、元組、字典等）中的每個元素。它是遍歷陣列最常用的方式。

基本語法

for 變數 in 可迭代對象: # 對每個元素執行的程式碼 程式碼區塊
注意：Python 使用縮排（通常是 4 個空格）來表示程式碼區塊。 在 for 迴圈中，所有縮排的程式碼都會對每個元素執行一次。

使用 for 迴圈遍歷陣列

numbers = [1, 2, 3, 4, 5] for num in numbers: print(num)

視覺化演示

numbers = [1, 2, 3, 4, 5] for num in numbers: print(num)

For 迴圈的特點

  • 簡潔易讀，特別適合遍歷已知序列
  • 自動處理迭代，不需要手動更新計數器
  • 可以使用 range() 函數生成數字序列
  • 同樣支持 break 和 continue 語句

使用 for 迴圈和 enumerate() 獲取索引

fruits = ['蘋果', '香蕉', '橘子', '葡萄', '芒果'] for index, fruit in enumerate(fruits): print(f"索引 {index}: {fruit}")

實際應用：修改陣列中的值

numbers = [1, 2, 3, 4, 5] # 將每個元素乘以 2 for i in range(len(numbers)): numbers[i] = numbers[i] * 2 print(numbers) # 輸出: [2, 4, 6, 8, 10]

in 關鍵字

in 是 Python 中一個非常重要的關鍵字，它在迴圈和條件判斷中都有重要作用。

在 for 迴圈中的 in

在 for 迴圈中，in 用於指定要遍歷的序列：

for 元素 in 序列: # 對每個元素執行的操作

例如：

fruits = ['蘋果', '香蕉', '橘子'] for fruit in fruits: print(fruit)

這段程式碼會依次輸出 "蘋果"、"香蕉" 和 "橘子"。

在條件判斷中的 in

in 也可以用於檢查一個元素是否存在於序列中：

if 元素 in 序列: # 如果元素在序列中，執行這裡的程式碼

例如：

fruits = ['蘋果', '香蕉', '橘子'] if '香蕉' in fruits: print('香蕉在水果列表中！')

這段程式碼會輸出 "香蕉在水果列表中！"，因為 '香蕉' 確實存在於 fruits 列表中。

in 關鍵字的其他用途

1. 遍歷字典

person = {'name': '張三', 'age': 30, 'city': '台北'} # 遍歷字典的鍵 for key in person: print(key) # 遍歷字典的值 for value in person.values(): print(value) # 遍歷字典的鍵值對 for key, value in person.items(): print(f"{key}: {value}")

2. 遍歷字符串

message = "Python" for char in message: print(char)

3. 與 range() 函數結合

# 產生 0 到 4 的數字序列 for i in range(5): print(i) # 輸出: 0, 1, 2, 3, 4 # 產生 1 到 10 的數字序列 for i in range(1, 11): print(i) # 輸出: 1, 2, 3, ..., 10 # 產生 0 到 10 的偶數序列 for i in range(0, 11, 2): print(i) # 輸出: 0, 2, 4, 6, 8, 10

小結

in 關鍵字是 Python 中非常靈活的工具，它可以：

  • 在 for 迴圈中用於遍歷序列中的每個元素
  • 在條件判斷中用於檢查元素是否存在於序列中
  • 適用於多種數據類型，包括列表、元組、字典、字符串等
  • 與 range() 函數結合使用，產生數字序列進行迭代

While 與 For 迴圈比較

特點 While 迴圈 For 迴圈
基本用途 當不確定迭代次數時使用 遍歷已知序列或集合
控制方式 需要手動更新條件 自動迭代
適用場景 基於條件的迭代 遍歷集合中的所有元素
風險 可能導致無限迴圈 通常不會有無限迴圈問題
代碼簡潔性 相對較長 通常更簡潔

何時使用 While 迴圈？

  • 當你不知道需要迭代多少次時
  • 當迭代需要基於某個條件而不是集合大小時
  • 需要在未知條件滿足前持續執行某個操作
  • 例如：等待用戶輸入特定值、直到找到特定元素為止

何時使用 For 迴圈？

  • 當你需要遍歷一個已知序列中的所有元素
  • 當你需要執行固定次數的操作
  • 當你需要同時獲取元素及其索引
  • 例如：處理列表中的每個元素、遍歷字典的鍵值對
